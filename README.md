## 📄 Projects

### `product-page.html` — AURA Headphones Landing Page

A fully responsive product landing page for a fictional premium headphone brand.

**Features:**
- Fixed navigation bar with blur backdrop
- Animated hero section with floating SVG product illustration
- Scroll-triggered reveal animations
- Technical specs grid
- Interactive color swatch selector
- 3-column pricing section
- Fully responsive (mobile-friendly)
- Pure HTML + CSS + vanilla JS — zero dependencies

**Preview sections:**
| Section | Description |
|---|---|
| Hero | Product headline, CTA buttons, animated illustration |
| Badge Strip | Key specs at a glance (battery, ANC, weight) |
| Specs | Grid layout with highlighted numbers |
| Colors | Clickable color swatches |
| Pricing | 3-tier pricing cards |
| Footer | Brand footer |

**How to run:**

No setup needed. Just open the file in any browser:

```bash
# Option 1 — double-click the file in your file explorer

# Option 2 — open from terminal (Mac/Linux)
open product-page.html

# Option 3 — open from terminal (Windows)
start product-page.html
```

**Built with:**
- Semantic HTML5
- CSS custom properties (variables)
- CSS Grid & Flexbox
- CSS animations & transitions
- Intersection Observer API (scroll reveal)
- Google Fonts (Bebas Neue + DM Sans)
